GitHub 雲端整合設定

設定後，學生在網頁上傳的圖片會直接寫入你的 GitHub Repo，且此網頁能直接讀取該資料夾。這可以達到「無資料庫」的全靜態雲端儲存效果。

教師提示：如何讓學生上傳不需重複輸入？

當你設定好 GitHub 資訊後，可以點擊下方「產生配置貼代碼」按鈕，將生成的配置區塊複製到 index.html 程式碼中。這樣所有學生打開此網頁，不需輸入金鑰即能直接上載！
